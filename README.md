
# 🎬 Movie Review App

A modern Movie Review Web Application built using **React JS** that allows users to search movies, view details, and give star-based ratings ⭐.

---

## 🚀 Features

✨ Search movies using API  
🎥 View movie details (Poster, Title, Year, Plot)  
⭐ Add star-based ratings  
🔍 Filter movies  
📱 Responsive UI (Mobile + Desktop)  
⚡ Fast performance with Vite  

---

## 🛠️ Tech Stack

- ⚛️ React JS
- ⚡ Vite
- 🎨 Tailwind CSS
- 🌐 Axios (API calls)
- 🔗 React Router DOM
